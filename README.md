# Roman Personal 🚀

O platformă personală modernă pentru management financiar, sănătate și recuperare.

## Funcționalități

- 📊 **Dashboard** — statistici live + chat AI
- 💰 **Finanțe** — categorii de cheltuieli, rapoarte lunare, arhivă, consultant AI
- 🏃 **Sănătate** — pași, apă, somn, calorii
- 🚫 **Stop Gambling** — streak, motivație AI, jurnal recuperare
- 📖 **Jurnal** — intrări cu mood tracker
- ✅ **Task Manager** — priorități, filtre
- 📚 **Studiu** — tutore AI
- 📁 **Fișiere** — cloud personal

## Instalare pe telefon (PWA)

1. Deschide aplicația în **Chrome** pe Android sau **Safari** pe iPhone
2. Apasă **Meniu → Adaugă pe ecranul principal**
3. Aplicația se instalează cu iconul RB și funcționează offline

## Deploy pe GitHub Pages

1. Creează un repository nou pe GitHub
2. Încarcă toate fișierele
3. Mergi la **Settings → Pages → Branch: main**
4. Accesează `https://username.github.io/repository-name`

## Tehnologii

- HTML / CSS / JavaScript pur
- Tailwind CSS (CDN)
- Google Fonts (Michroma + Inter)
- Claude AI (Anthropic API) pentru funcțiile AI
- localStorage pentru persistența datelor
- PWA cu Service Worker pentru instalare offline
